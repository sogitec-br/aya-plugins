# Política de segurança

## Como relatar

Não abra uma issue pública para vulnerabilidades, falhas de autorização, exposição de dados ou
segredos. Utilize o recurso **Report a vulnerability** na aba Security deste repositório:

<https://github.com/sogitec-br/aya-plugins/security/advisories/new>

Inclua somente o mínimo necessário para reprodução. Nunca envie senha, access token, refresh token,
cookie, header de autorização ou dados reais de clientes.

## Versões suportadas

| Versão | Suporte |
|---|---|
| `0.1.x` | Beta, correções de segurança quando aplicáveis |

## Escopo

Este repositório contém o cliente/plugin. Problemas no serviço MCP ou no SOGI devem ser relatados
pelo mesmo canal privado para triagem e encaminhamento.
