# Analise SOGI

1. Obtenha os dados pelo workflow de descoberta e execucao; nao use amostras antigas como atuais.
2. Preserve o payload original e derive agregacoes de forma reproduzivel.
3. Informe denominadores, filtros, periodo e campos usados em percentuais ou rankings.
4. Separe fatos retornados, calculos derivados e hipoteses explicativas.
5. Nao declare causalidade a partir de correlacao ou de um unico recorte.
6. Minimize dados pessoais e use agregacoes sempre que atenderem ao pedido.
7. Quando uma visualizacao ajudar, escolha a forma mais simples: tabela para detalhe, barras para
   comparacao, linha para tempo e distribuicao para frequencia.

Leia `../../references/governance.md` antes de produzir um artefato com dados granulares.
